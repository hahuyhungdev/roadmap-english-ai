# BUỔI 4: Tools, Technologies & Tech Stack

**Level:** B1+  
**Focus:** Describe the main tools and technologies you use as a Frontend Developer in a simple, clear way with business and user benefits.

## 1) Vocabulary (with 2–3 useful examples each)

- **technology** (n) – tools, languages, or frameworks used to build software
  - React is one of the main technologies I work with.
  - Modern technologies help us build faster and better applications.
  - We choose technologies that are reliable and easy to maintain.

- **framework** (n) – a set of tools that makes development easier
  - We use React as our main frontend framework.
  - A good framework saves development time and improves consistency.
  - The framework we chose supports responsive design very well.

- **library** (n) – a collection of reusable code
  - We use several UI libraries to speed up development.
  - Popular libraries help us create consistent buttons and forms.
  - I often check the documentation of the libraries I use.

- **version control** (n) – a system to track changes in code
  - We use Git for version control.
  - Good version control allows the team to work together safely.
  - Version control helps us go back to previous versions if needed.

- **performance** (n) – how fast and efficient something is
  - We always focus on improving application performance.
  - Better performance leads to happier users.
  - Poor performance can cause users to leave the application.

- **responsive** (adj) – works well on different screen sizes
  - We make sure every page is fully responsive.
  - Responsive design is important for mobile users.
  - A responsive interface improves the overall user experience.

## 3) Typical Dialogues

### Dialogue 1 – Simple Explanation of Tech Stack

**Teacher:** What technologies and tools do you use?

**You:**  
As a frontend developer, I mainly work with React, which is a popular JavaScript framework. We also use TypeScript to write safer code and Tailwind CSS for styling. For version control, we use Git and GitHub.

These tools help us build modern, fast, and responsive user interfaces. We also use Figma to see designs and Jira to manage our tasks.

### Dialogue 2 – Explaining Benefits & Why You Use Them

**Teacher:** Why do you use React instead of other frameworks?

**You:**  
We chose React because it allows us to build reusable components, which makes development faster and the code easier to maintain. It also helps create smooth and responsive user interfaces.

For example, when we need to update one part of the application, we can do it without affecting other parts. This saves time and reduces the chance of bugs. In the end, better tools lead to better user experience and faster delivery of new features.

### Dialogue 3 – Handling a Simple Objection

**Teacher:** Aren’t there too many tools to learn?

**You:**  
It may seem like a lot at first, but each tool has a clear purpose. Once you learn them, they make the work much easier and more efficient. Good tools help the team collaborate better and deliver higher quality work in less time.

## 4) Reading Text

In my work as a frontend developer, I use a modern technology stack that includes React, TypeScript, and Tailwind CSS. These technologies help us create fast, responsive, and maintainable user interfaces.

We also use tools like Figma for design collaboration, Git for version control, and Jira for task management. The combination of these tools allows the team to work efficiently, keep the code clean, and deliver new features quickly.

By using the right technologies, we can focus more on solving user problems and improving the overall experience instead of struggling with basic development tasks.
