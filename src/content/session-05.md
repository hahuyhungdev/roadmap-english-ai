# BUỔI 5: Main Features & Functionalities

**Level:** B1+  
**Focus:** Describe the most important features of the system clearly, using simple language and emphasizing user benefits and business value.

## 1) Vocabulary (with 2–3 useful examples each)

- **feature** (n) – a specific function or capability of the system
  - One of the most useful features is the real-time dashboard.
  - We recently added a new search feature to improve usability.
  - Each feature is designed based on user feedback.

- **functionality** (n) – the set of functions that the system can perform
  - The core functionality allows users to manage orders easily.
  - This functionality saves a lot of manual work.
  - We are constantly improving the existing functionality.

- **dashboard** (n) – a visual display of important information
  - The main dashboard shows key metrics at a glance.
  - Users can customize their personal dashboard.
  - A clear dashboard helps managers make quick decisions.

- **real-time** (adj) – happening immediately, without delay
  - The system provides real-time updates on order status.
  - Real-time notifications keep users informed.
  - Real-time data helps teams respond faster to problems.

- **search** (n/v) – the ability to find information quickly
  - The advanced search feature makes finding records easy.
  - Users can search by multiple filters.
  - Good search functionality improves productivity.

- **notification** (n) – a message that alerts the user
  - The system sends automatic notifications when something changes.
  - Push notifications help users stay updated on mobile.
  - We reduced unnecessary notifications to avoid alert fatigue.

- **workflow** (n) – the sequence of steps to complete a task
  - The new workflow makes approval processes much faster.
  - We simplified the workflow to reduce errors.
  - A smooth workflow improves overall efficiency.

## 3) Typical Dialogues

### Dialogue 1 – Explaining Key Features

**Teacher:** What are the main features of the system?

**You:**  
The system has several important features. The main dashboard gives users a quick overview of key metrics and current status. There is also a powerful search function that lets users find information quickly using different filters.

Another key feature is real-time notifications. When an order status changes or a task needs attention, users receive instant alerts. These features together help users work more efficiently and stay informed at all times.

### Dialogue 2 – Business & User Impact

**Teacher:** How do these features help the business?

**You:**  
These features have a direct impact on productivity and accuracy. For example, the real-time dashboard allows managers to see problems immediately instead of waiting for daily reports. The streamlined workflow reduces the time needed to approve orders by almost half.

As a result, the company can process more orders with fewer errors, which improves customer satisfaction and saves operational costs.

### Dialogue 3 – Handling a Question about Complexity

**Teacher:** Aren’t there too many features? Doesn’t it make the system complicated?

**You:**  
We design the features to be simple and user-friendly. Even though there are many functions, users only see what they need. We also provide good search and filtering options so people can find things quickly. The goal is to make work easier, not more complicated.

## 4) Reading Text

The system offers many useful features that support daily business operations. The central dashboard displays important information such as order status, inventory levels, and performance metrics in one place. Users can search for records quickly with multiple filters and receive real-time notifications when important events occur.

There is also a streamlined approval workflow that reduces manual steps and minimizes errors. All these functionalities are built with a clean and responsive interface so that users can access them easily from any device. These features help teams work faster, make better decisions, and deliver higher quality service to customers.
